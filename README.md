# Description

The chess manager provides a menu to handle a tournament of chess. 

# Prerequisites

Python 3

# Installation

```
pip install virtualenv
virtualenv venv
pip install -r requirements.txt
```

# Start

```
python main.py
```
# Code Style
```
flake8 .
```
To get a report
```
flake8 --format=html --htmldir=flake8-report
```
